# SUS-ADB 1.0.0-rc.1

This release candidate consolidates the authorized Android assessment workspaces, adds user-local configuration, redacted local diagnostics, crash recovery, reproducible packaging definitions, CI validation, and release documentation. Script Studio, its user-local Script Library, and import/edit/validate/explicit-load workflows are included, but RC1 does not bundle a reviewed core curated script pack. The disabled hello example plugin includes a harmless demonstration asset. User-authored and third-party scripts require explicit review and approval, and private/local Script Studio files are excluded from release packages.

RC1 does not create a release tag, publish binaries, bundle external tools, upload telemetry, or alter user devices automatically.
