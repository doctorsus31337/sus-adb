# sus-adb

## Android Device Companion

sus-adb is a modular cross-platform companion application
for Android device management, debugging, and instrumentation.

## Features

Planned:

- ADB management
- Frida integration
- Objection integration
- Device profiles
- Workspace management
- Plugin architecture
- Gothic themed GUI

## Supported Platforms

- Windows
- Linux
