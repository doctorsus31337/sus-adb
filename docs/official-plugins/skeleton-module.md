# Skeleton Module

Skeleton is a disabled, zero-capability, no-op educational package. Its digest requires explicit **Trust**, never a misleading Permissions grant. **Export Template…** creates a bounded editable developer copy through a host-owned destination chooser without installing or executing it.

Before installing a derivative, change the official plugin ID, display name, and version, then review every capability request. Editing an exported copy does not modify the bundled Skeleton. The copy remains untrusted, disabled, unloaded, and uninstalled until explicitly submitted through Plugin Manager. Its comments and tutorial explain lifecycle, façades, scope, contributions, workers, cancellation, Tk marshalling, structured results, digest invalidation, tests, packaging, and uninstall.
