# Rootability Advisor

The advisor parses safely observable identity/boot/root state and bounded metadata from selected firmware inputs. Results are passed checks, blockers, warnings, unknowns, likely wipe consequences, prerequisites, previews, references, and rollback guidance. Compatibility requires evidence. It never searches for exploits, executes ADB/Fastboot, downloads, roots, patches, unlocks, flashes, wipes, or changes verified boot/anti-rollback.
