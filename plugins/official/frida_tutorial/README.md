# Frida Assistant

An official, disabled-by-default contextual helper. It requests only immutable
selected-device and selected-target state. Host-owned actions delegate to the
shared discovery, diagnostics, Sessions Center, and Script Studio services.
Opening the assistant performs no scan, attach, spawn, server start, script
load, APK change, device modification, terminal launch, or network action.

The original Frida Foundations course remains available under **Learn**, with
synthetic examples and harmless optional practice.
