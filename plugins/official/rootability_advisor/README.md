# Rootability & Bootloader Readiness Advisor

This plugin parses read-only observations and safely inspects operator-selected firmware headers or bounded archive metadata. It produces evidence-based compatibility classifications, blockers, warnings, unknowns, data-wipe guidance, and command previews. It never executes ADB or Fastboot itself, downloads firmware, exploits, unlocks, patches, flashes, wipes, hides root, or changes verified boot or anti-rollback state.
