# Device Rescue & Recovery

This plugin plans and performs bounded, explicitly selected copying through injected authorized access. It has no lock, FRP, decryption, exploit, root-acquisition, unlock, flash, wipe, deletion, upload, or persistence behavior. Public presets cover common shared-storage folders only. Private paths require existing root, active matching scope, visible limits, and confirmation. `scrcpy`, MTP, OTG, and USB-C display information is guidance only.
