# Step-by-step tutorial

1. Copy this directory outside the bundled catalog. 2. Choose a unique stable lowercase ID. 3. Keep API version `1.0`. 4. Increment semantic versions when behavior changes. 5. Declare platforms, every capability, entry point, contribution, and `addon_ui.ui_mode` (`embedded`, `window`, or `hybrid`). 6. Implement immutable models and structured results. 7. Add bounded cancellation-aware workers for long work. 8. Request host callback marshalling rather than retaining Tk. 9. Test with fakes. 10. Reinstall and reapprove after any digest change. The host owns windows, singleton behavior, theme, geometry, closing, and cleanup.
